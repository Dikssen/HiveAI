# Contributing to HiveAI

Thank you for your interest in contributing to HiveAI.

HiveAI is an AI-powered multi-agent operations platform for IT teams. Contributions are welcome, including bug reports, feature ideas, documentation improvements, and code changes.

## Before You Start

Please check existing issues and pull requests before opening a new one.

For larger changes, open an issue first to discuss the idea.

## Development Setup

1. Fork the repository.
2. Clone your fork:

```bash
git clone https://github.com/Dikssen/HiveAI.git
cd HiveAI
