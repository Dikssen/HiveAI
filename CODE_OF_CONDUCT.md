# Code of Conduct

## Our Pledge

We want HiveAI to be a welcoming and respectful project for everyone.

All contributors, maintainers, and users are expected to communicate respectfully, assume good intent, and help keep discussions productive.

## Expected Behavior

Examples of positive behavior include:

- Being respectful and considerate
- Giving constructive feedback
- Focusing on what is best for the project
- Helping newcomers understand the project
- Accepting different opinions and experience levels

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Trolling or intentionally derailing discussions
- Publishing someone else's private information
- Spam or unrelated promotion

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or block users who violate this Code of Conduct.

If you see behavior that violates this Code of Conduct, please contact the maintainer privately through GitHub.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any other public communication related to HiveAI.
